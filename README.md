# Engine Build Room #1 — Interactive Floor Map

Interactive floor map for Engine Build Room #1 at Tenneco.  
Tap a cabinet to see what's inside.

## How to Update

Edit `index.html` — the room data is in the `ROOM` object near the top of the `<script>` block.  
Change items, add cabinets, update statuses there.

## Hosting

This is deployed via GitHub Pages. Any push to `main` auto-publishes.
